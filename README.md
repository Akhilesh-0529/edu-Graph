# Intelligent Educational Q&A System

A full-stack simulation of an AI-powered educational Q&A system built with vanilla **HTML**, **CSS**, and **JavaScript**.

## Key Features Built In

- **Interactive Chat Interface**: Write questions (e.g. *What is Calculus?* or *Explain Derivatives*) and see a vector similarity RAG engine retrieve relevant document context in real time.
- **Dynamic Knowledge Graph**: View visual SVG concept relationship maps (Math -> Calculus -> Derivatives -> Integrals) and click on concepts to query them.
- **Roles Switcher**: Seamlessly switch between **Student, Teacher, and Administrator** views in the header.
- **Teacher workspace**: Upload files (emulated ingestion) and resolve escalated student queries.
- **Interactive Student Dashboard**: View learning stats and concept mastery bars.
